DDl commands(data definition languange):
DDl (data definition languange) refers to SQL commands that are used to
define, modify, or delete the structure of database objects
such as tables,view,indexes and triggers

C-create
A-alter
T-truncate
D-drop
R-rename

1.create:
the create command is used to create new database objects like
tables,views, indexes or triggers

Syntax:
CREATE TABLE table_name(
    column1 datatype constraint,
    column2 datatype constraint,
    ..................................
);

example:

CREATE TABLE student (
  rollno INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  dept_id INTEGER
);



---------------------------------------------------------------
2.Alter command

the Alter command is used to modify the structure of an existing table
such as adding or renaming a column or renaming a table

Synatx:

ALTER TABLE table_name ADD COLUMN age integer;

example:
alter table student rename to students;

if it is column name:
alter table table_name rename column name to new_colmun_name

-------------------------------------------------------------------------

3.Drop command:

the drop command is used to permanently delete an existing

5.RENAME command
the rename command is used to change the name of an exsting table or
column.

Synatx:
